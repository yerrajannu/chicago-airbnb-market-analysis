# Chicago Airbnb Market Pricing & Demand Analysis

## Overview
Analyzed Chicago Airbnb listings to identify mismatches between listing 
prices and guest demand across neighborhoods using Tableau.

## Tools
Tableau | 2 Datasets | 1,000+ Listings

## Key Findings
- Designed a Pricing Efficiency Index (Price ÷ Reviews/Month) to quantify 
  price-demand mismatch across neighborhoods
- Peripheral neighborhoods like Mount Greenwood outperformed central areas 
  in guest demand despite lower prices
- Built spatial map dashboard encoding demand (marker size) and pricing 
  efficiency (diverging color) simultaneously
